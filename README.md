# Air India Airlines Flight Booking System

This project is a **console-based Flight Booking System** for Air India Airlines. It allows users to manage flight bookings, add customer details, view ticket prices, and print tickets. The system is built with C++ and runs on the terminal, providing a user-friendly experience for booking flights.

---

## Features

### **Main Menu**
The system provides a menu-driven interface with the following options:
- Add customer details.
- Register for flights.
- Check ticket prices and charges.
- Exit the system.
